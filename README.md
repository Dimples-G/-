# 简易网页计算器
## 1.功能
- 支持简单的加减乘除功能
- 支持小数、整数
- 支持连续符号输入取后者
- 支持单纯符号输入判错提示、除数为 0 判错提示
## 2.快速开始
1. 下载 `calculator.html` `calculator.css` `calculator.js` 在同一个文件夹下
2. 双击 `calculator.html` 快速开始
## 3.文件说明
| 文件名     |   作用  |
|:-------------|:-------------|
|calculator.html|页面结构|
|calculator.css|样式设计、布局设置|
|calculator.js|计算逻辑|
